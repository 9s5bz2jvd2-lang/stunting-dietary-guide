# 食养方（按证型分类）

> 基于《儿童青少年生长迟缓食养指南（2023年版）》附录4，以8岁学龄儿童为例

---

## 健脾消食类

### 一、山楂麦芽消食汤

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃气虚证（食积、消化不良） |
| **主要材料** | 山楂5g，麦芽15g，干山药15g，橘皮2g，猪瘦肉100g |
| **制作方法** | 山楂、麦芽、干山药、橘皮清洗去浮尘，浸泡30分钟。猪瘦肉洗净切块。所有食材放入锅内，加适量清水，大火滚沸后改中小火再煮30分钟，调味即可 |
| **用法用量** | 餐前30~60分钟喝1小碗汤，每周2~3次 |

### 二、莲子芡薏健脾粥

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃气虚证、脾虚湿困证 |
| **主要材料** | 干莲子15g，干芡实15g，薏苡仁15g，粳米25g |
| **制作方法** | 干莲子去芯，薏苡仁炒制。干莲子、干芡实、炒薏苡仁浸泡30分钟。粳米淘洗干净，所有材料放入锅内，加水煮熟 |
| **用法用量** | 代替部分主食，每周2~4次 |

### 三、山楂鸡内金茯苓饼

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃气虚证、脾虚湿困证 |
| **主要材料** | 山楂10g，鸡内金20g，茯苓30g，山药粉30g，黑白芝麻各20g，面粉500g |
| **制作方法** | 山楂、鸡内金、茯苓研磨成粉，与山药粉、面粉混合均匀，加水和面，擀成小薄面饼，撒芝麻，烤箱烤熟 |
| **用法用量** | 零食，餐后加餐，每次2~3块 |

### 五、板栗山药粥

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃气虚证、脾胃阴虚证 |
| **主要材料** | 鲜山药100g，板栗5颗，大枣3颗，粳米50g |
| **制作方法** | 板栗煮熟，大枣去核对半切开，山药去皮切块。粳米煮开后倒入板栗、大枣，小火煮30分钟。加入山药，小火煮30分钟 |
| **用法用量** | 代替部分主食，每次1碗，每周2~3次 |

### 八、牛奶山药麦片粥

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃气虚证、脾胃阴虚证 |
| **主要材料** | 牛奶100ml，干山药20g，薏苡仁15g，麦片25g，大枣2颗 |
| **制作方法** | 薏苡仁炒制，大枣去核对半切开，山药切小丁。锅中加水放麦片和薏苡仁，大火煮开。加入山药、大枣同煮至浓稠，倒入牛奶煮5分钟 |
| **用法用量** | 代替部分主食，每周2~3次 |

## 健脾益肾类

### 三、核桃桂圆益智粥

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃气虚证、肾阳虚证 |
| **主要材料** | 核桃仁5个，干桂圆5g，益智仁5g，粳米50g |
| **制作方法** | 核桃去壳留仁。干桂圆、益智仁冲洗。粳米淘洗后加水，所有材料放入煮熟 |
| **用法用量** | 代替部分主食，每周2~3次 |

### 四、枣仁百合排骨汤

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃阴虚证、肾阴虚证 |
| **主要材料** | 酸枣仁5g，干百合10g，大枣2颗，猪排骨100g |
| **制作方法** | 酸枣仁砸碎可装小汤袋。排骨洗净斩小段焯水。干百合洗净，大枣去核对半切开。所有材料放入锅中，加水炖煮约1小时，调味 |
| **用法用量** | 晚餐佐餐或睡前1碗汤，每周2~3次 |

### 六、麻仁杏仁猪肉汤

| 项目 | 内容 |
|------|------|
| **适应证型** | 脾胃阴虚证（肠燥便秘） |
| **主要材料** | 火麻仁5g，甜杏仁10g，干无花果2个，猪肉100g |
| **制作方法** | 火麻仁、甜杏仁、干无花果清洗备用。猪肉切块焯水洗净。所有食材放入锅内，加水，大火煮沸后改中小火煮40分钟，调味 |
| **用法用量** | 餐后1碗，连用2~3天 |

## 温补益肾类

### 九、枸杞核桃炖羊肉

| 项目 | 内容 |
|------|------|
| **适应证型** | 肾阳虚证 |
| **主要材料** | 羊肉100g，核桃仁5个，枸杞子3g，生姜5片，胡椒粉适量 |
| **制作方法** | 羊肉切小块焯水洗净。枸杞子、核桃仁洗净。砂锅放羊肉、枸杞子、核桃仁，加水，加胡椒粉、葱姜片，烧开去浮沫，转小火炖至肉烂 |
| **用法用量** | 佐餐食用，每周1~2次 |

---

> ⚠️ 注意事项：所有食养方以8岁学龄儿童为例，其他年龄段份量需调整。建议在医师或营养指导人员指导下选择使用。有相应症状者可食用至症状改善或消失。
