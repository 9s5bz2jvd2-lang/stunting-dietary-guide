# 营养学 | 儿童青少年肥胖食养助手

基于国家卫生健康委《儿童青少年肥胖食养指南（2024年版）》及官方问答蒸馏的 AI 营养科普对话 Skill。

创建者：王润圆（中国注册营养师，昆明医科大学营养与食品卫生学硕士）  
协作构建：Work Buddy和灵台AI一起构建

## 内容范围

- `SKILL.md`：Skill主提示词，含四维配置、关键数据速查、场景化应答、边界声明。
- `knowledge_base.md`：14个KPK科普知识点，覆盖肥胖判定、6大食养原则、食物选择、食物交换表、地区特点、营养方和官方问答。
- `recipes_data.md`：5大地区食谱示例，共24天样例。
  - 东北：冬春3天 + 夏秋3天
  - 西北：冬春3天 + 夏秋3天
  - 中部：冬春3天 + 夏秋3天
  - 西南：3天示例
  - 东南：3天示例
- `dietary_formulas.md`：10个营养方（5个辨证营养方 + 5个常见症状营养方）。
- `skill.yaml`：元数据。
- `system_prompt.md`：可直接用于对话助手的系统提示词版本。

## 使用场景

当用户提到儿童肥胖、孩子体重超标、BMI超标、儿童饮食控制、儿童减重、屏幕时间、运动、校餐、家庭饮食调整、中医证型（胃热火郁/痰湿内盛/气郁血瘀/脾虚不运/脾肾阳虚）等话题时触发。

## 适用范围与不适用范围

- **适用对象**：基层卫生工作者（含营养指导人员）、2~17岁儿童青少年家长、校医。
- **适用情境**：以饮食营养为主的儿童青少年单纯性肥胖的预防、控制和日常食养指导。
- **不适用**：
  - 由内分泌、遗传、神经系统等疾病引起的继发性肥胖的诊断与治疗；
  - 0~2岁婴幼儿喂养；
  - 重大合并症（详见"安全边界"）情况下的个体化食疗，需先经医院评估。

## 安全边界

- 不做医学诊断；不替代医生治疗与注册营养师/营养指导人员的个体化方案。
- 儿童青少年正在生长发育，不推荐盲目节食或极端减重。
- 单纯性肥胖儿童青少年不建议药物和手术治疗。
- **重大合并症安全红线**：出现肾功能异常、eGFR/GFR异常、尿蛋白阳性、血尿、疑似或已诊断慢性肾脏病（CKD），以及心力衰竭、肝功能衰竭、严重心律失常等重要脏器功能不全时，**不直接给出个体化食养方案**，应先到医院评估，由医生与注册营养师/营养指导人员制定方案后，本资料仅作为日常执行参考；蛋白质、食盐、钾、磷及总能量摄入目标可能需医生根据病情调整。
- 重度肥胖或伴有其他代谢性疾病者，应在专业人员指导下制定方案。
- 食药物质和营养方应在医生或营养指导人员等专业人员指导下使用。
