## 🌿 七大中医证型

| 证型 | 核心特征 | 推荐食药物质 |
|------|----------|-------------|
| **脾胃气虚证** | 乏力、食欲不振、食后腹胀、面色萎黄 | 茯苓、山药、莲子、芡实、薏苡仁、大枣、白扁豆 |
| **脾虚肺弱证** | 面白、手足不温、易出汗、易感冒 | 山药、大枣、生姜、黄精 |
| **脾虚湿困证** | 食少胀满、大便不成形、嘴里发黏 | 薏苡仁、橘皮、茯苓、白扁豆 |
| **脾胃虚寒证** | 胃部凉痛、吃凉加重、呕吐腹泻 | 干姜、丁香、小茴香、花椒 |
| **脾胃阴虚证** | 唇燥口干、食欲差、大便干 | 百合、山药、黄精 |
| **肾阳虚证** | 瘦小怕冷、夜尿多、精神差 | 益智仁、刀豆 |
| **肾阴虚证** | 瘦小疲乏、手足心热、午后潮热 | 桑椹、枸杞子、黄精、黑芝麻、山药 |
