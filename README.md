## 指南出处

- **全称**：《儿童青少年生长迟缓食养指南（2023年版）》
- **发布单位**：国家卫生健康委办公厅

## 功能特点

- **身高判定标准**：2~5岁+6~17岁完整身高对照表，含第3百分位切点
- **食养原则**：6条官方原则，围绕生长发育核心需求
- **中医食养**：7种证型（脾胃气虚/脾虚肺弱/脾虚湿困/脾胃虚寒/脾胃阴虚/肾阳虚/肾阴虚）+ 9个食养方
- **地区食谱**：5大地区×每地区3天食谱示例
- **食物交换表**：7类食物等量交换，灵活配餐
- **生长监测指导**：定期测量身高体重，绘制生长曲线
- **科普风格**：说人话、给具象量、指误区——精准但不端着

## 关键数据速查

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 生长迟缓判定（2-5岁） | 身高<同年龄同性别第3百分位 | 比97%的同龄孩子矮 |
| 生长迟缓判定（6-17岁） | 身高<同年龄同性别第3百分位 | 同上，需及时干预 |
| 蛋白质（学龄儿童） | 足量优质蛋白 | 每天一杯奶+一个蛋 |
| 奶制品 | ≥300mL/天 | 一杯奶打底 |
| 维生素D | 400 IU/天 | 日晒不足时需补充 |
| 户外活动 | ≥1小时/天 | 阳光+运动双管齐下 |
| 睡眠（学龄儿童） | 9-11小时/晚 | 生长激素夜间分泌最旺盛 |
