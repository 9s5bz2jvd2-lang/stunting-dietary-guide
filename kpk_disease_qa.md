> 基于《儿童青少年生长迟缓食养指南（2023年版）》前言、疾病特点与分型及问答版

## KPK-12：疾病背景、定义与中医分型

### 核心知识

生长迟缓是指身高低于同年龄、同性别正常身高标准参照值，反映长期膳食营养失衡，是营养不良的一种类型。

### 适用人群

2~17岁儿童青少年及家长、基层卫生工作者

### 一、定义与判定

- **定义**：由于膳食的蛋白质或能量、维生素、矿物质摄入不足等原因，导致身高低于筛查标准的年龄别身高界值范围，属长期性营养不良
- **判定标准**：
  - 2~5岁儿童：依据《7岁以下儿童生长标准》（WS/T 423），身高低于同年龄同性别2个标准差
  - 6~17岁儿童青少年：依据《学龄儿童青少年营养不良筛查》（WS/T 456），身高小于或等于相应界值范围
  - 详见KPK-11表5.1、表5.2

### 二、流行病学

- 6岁以下儿童生长迟缓率：**4.8%**
- 6~17岁儿童青少年生长迟缓率：**1.7%**
- 城乡差异明显：农村约为城市的**2~3倍**
- 数据来源：《中国居民营养与慢性病状况报告（2020年）》

### 三、影响与危害

- 儿童青少年期：体格和智力发育受影响，体能下降、学习表现不佳
- 严重时：容易导致腹泻、肺炎等疾病，增加感染性疾病发病率和死亡率
- 成年后：可能影响身高，增加肥胖、心血管疾病、糖尿病等慢性病发病风险，降低劳动生产能力

### 四、病因

- 膳食结构不合理
- 不良饮食行为
- 家长喂养不当
- 中医理论：长期食欲不振、厌食、饮食积滞、节食等，造成脾胃运化失调

### 五、适用范围与排除

**适用**：
- 主要针对膳食营养相关的生长迟缓（蛋白质—能量营养不良，维生素、矿物质缺乏等）
- 有合并症者可参考（但不拘泥于）本指南

**不适用**（需就医）：
- 食物过敏导致的生长迟缓
- 特发性矮小症
- 生长激素缺乏或功能缺陷
- 慢性系统性疾病等病理性原因

### 六、中医对生长迟缓的认识

中医认为儿童"脾常不足"，脾胃功能全而未壮。脾胃为后天之本，气血生化之源；肾藏精，主骨，生髓，为先天之本。脾胃和肾功能强弱与儿童青少年生长发育密切相关。

#### 七种辨证分型

| 证型 | 主要临床表现 |
|------|-------------|
| **脾胃气虚证** | 乏力，食欲不振，食后易腹胀，头晕，易困倦，精神不振，面色萎黄，舌淡，舌苔白，脉弱 |
| **脾虚肺弱证** | 面色苍白，手足不温，食少，大便不成形，短气，易出汗，易感冒，舌淡，舌苔白，脉细弱 |
| **脾虚湿困证** | 食少，胃部胀满不适，大便不成形，嘴里发黏，身体沉重感，懒言少动，舌淡红，舌苔厚，脉濡 |
| **脾胃虚寒证** | 胃部经常感觉凉、有痛感，吃寒凉东西后更加明显，腹胀，呕吐，食少，大便不成形，或经常拉肚子，乏力，易出现身体消瘦，舌淡，舌苔白，脉虚缓 |
| **脾胃阴虚证** | 唇燥口干，易口渴，嘴里感觉没滋味，食欲差，饭量逐渐减少，大便干不容易排出，舌红，舌苔少，或没有舌苔，脉细数 |
| **肾阳虚证** | 身材瘦小，体寒，怕冷，手脚凉，夜尿频多，白天易没有精神，后背感觉凉，大便稀或经常拉肚子，舌淡，舌苔薄白，脉沉迟无力 |
| **肾阴虚证** | 身材瘦小，易疲乏无力，耳鸣，易口舌干燥，咽干，手足心热，午后潮热，舌红，舌苔少，脉细数 |

## KPK-13：常见问答（Q1~Q9）

### Q1：《指南》制定的目的和意义是什么？

《儿童青少年生长迟缓食养指南（2023年版）》以满足人民健康需求为出发点，为预防和控制生长迟缓发生发展，帮助培养健康饮食习惯，提高营养健康水平，发展传统食养服务等提供科学指导。

**意义**：
1. 充分发挥现代营养学和传统食养的中西医各方优势，将食药物质融入合理膳食
2. 针对不同地区、不同季节编制食谱示例，提升膳食指导个性化和可操作性
3. 推动中西医结合，传承传统食养文化，推进食养服务高质量发展

### Q2：《指南》制定的依据是什么？

依据《健康中国行动（2019—2030年）》和《国民营养计划（2017—2030年）》相关要求。

### Q3：《指南》制定的原则是什么？

遵循四个原则：
1. **科学性原则**：以营养科学、中医食养理论、食物和健康关系证据为依据
2. **协调一致性原则**：确保与现行食品相关法律、法规协调一致
3. **公开透明原则**：广泛征求多方面意见
4. **指导性与实用性相结合的原则**：提出食养原则，列出可供选择的食物推荐、食谱和食养方

### Q4：国内外相关共识、指南和标准等情况如何？

参考了《中国学龄儿童膳食指南（2022）》、《学生餐营养指南》（WS/T 554—2017）、《学龄儿童体重管理营养指导规范》（T/CNSS 011—2021）、《儿童体格发育评估与管理临床实践专家共识》，以及2017年英国NICE发布的《儿童生长迟缓的识别和管理》指南等。

### Q5：《指南》的主要内容是什么？

给出食养原则和建议，附录包括：食物选择和食养方举例、不同证型食药物质推荐、不同地区不同季节食谱示例、常见食物交换表等工具。每个地区至少列举3天食谱（含食药物质）。

### Q6：《指南》发布后如何使用？

**面向对象**：基层卫生工作者（包括营养指导人员）、2~17岁儿童青少年的家长和校医等

**注意**：
- 主要针对膳食营养相关的生长迟缓
- 不可替代药物治疗
- 病理性原因导致的生长迟缓需在医师和营养指导人员指导下合理膳食

### Q7：生长迟缓儿童青少年在膳食上需要注意什么？

食物选择应全面、均衡、多样：
- 一日三餐、定时定量、饮食规律
- 每餐包括谷薯类、蔬菜水果、畜禽鱼蛋、奶和大豆等3类及以上
- 每天食物种类12种以上，每周25种以上
- 尤其重视高营养素密度食物：鱼虾类、深绿色叶菜、蛋黄等
- 可针对不同证型选用食药物质

### Q8：《指南》中的食养方如何使用？

- 食养方中的食药物质为国家卫生健康委公布的既是食品又是药品的物品
- 以8岁学龄儿童为例，其他地区、年龄的份量需调整
- 有相应症状的儿童青少年可食用至症状改善或消失
- 建议在医师、营养指导人员指导下选择和调整

### Q9：如何参考食谱示例，合理搭配日常膳食？

- 以8岁学龄儿童3天带量食谱为示例
- 使用《常见食物交换表》替换部分食物（如大米换小米、鸡肉换鸭肉、猪肉换牛肉、鱼肉换虾肉）
- 记住同类食物互换即可
- 其他年龄段根据表1.1不同年龄段食物建议摄入量进行适当调整
